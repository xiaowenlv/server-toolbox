# 🛠️ 服务器全能工具箱

一键管理 VPS 常用服务的 Bash 脚本，支持 Docker、代理面板、内网穿透、AI 服务等。

## 🚀 快速安装

```bash
# 一键安装快捷命令
bash <(curl -fsSL https://raw.githubusercontent.com/xiaowenlv/server-toolbox/main/install_tools.sh)

# 或者下载后运行
wget -O install_tools.sh https://raw.githubusercontent.com/xiaowenlv/server-toolbox/main/install_tools.sh
chmod +x install_tools.sh
./install_tools.sh
```

安装后直接输入 `sbox` 即可打开工具箱。

## 📋 功能一览

### 基础环境 & 双栈网络
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | Docker | 基础容器引擎，一键安装 |
| 2 | WARP | 添加虚拟 IPv6，解锁更多服务 |

### 代理与内网穿透
| 编号 | 功能 | 说明 |
|------|------|------|
| 3 | X-UI | 多协议代理面板（支持 Xray/V2Ray） |
| 4 | Lucky | 端口转发 / 反向代理 |
| 5 | FRP | 高速专属内网穿透 |

### AI 智能与协同工具
| 编号 | 功能 | 说明 |
|------|------|------|
| 6 | TeamTalk | 企业级语音服务（TT5SRV） |
| 7 | Hermes | 爱马仕 - 中文智能体控制台 |

### 其他
| 编号 | 功能 | 说明 |
|------|------|------|
| 8 | 创建快捷命令 | 生成 `sbox` 全局命令 |
| 0 | 退出 | 退出工具箱 |

## 🎯 使用方式

运行 `sbox` 后，按数字键选择功能：

```
       服务器全能工具箱 v2.2        
=================================================
 [基础环境 & 双栈网络]
   1. Docker    (基础容器引擎)
   2. WARP      (添加虚拟 IPv6)
 [代理与内网穿透]
   3. X-UI      (多协议代理面板)
   4. Lucky     (端口转发/反代)
   5. FRP       (高速专属内网穿透)
 [AI 智能与协同工具]
   6. TeamTalk  (企业级语音服务)
   7. Hermes    (爱马仕-中文智能体控制台)
=================================================
   8. 创建/更新快捷命令 (sbox)
   0. 退出工具箱
```

每个子服务都有独立管理菜单，支持：
- 一键部署（安装）
- 启动 / 重启
- 停止
- 查看登录信息（用户名、密码、端口）

## ⚠️ 注意事项

- 需要 root 权限运行
- 需要已安装 Docker（工具箱可一键安装）
- TeamTalk 和 Hermes 需要先部署才能查看登录信息
- 首次运行会自动创建 `sbox` 快捷命令
